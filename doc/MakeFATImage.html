MakeFATImage

MakeFATImage.pm is a perl module that can be used to create FAT file systems. It exports the following symbols:

  • MakeFATImage(file_name, label, cluster_size)

Usage

use MakeFATImage;

Description

  • MakeFATImage(file_name, label, cluster_size)

    MakeFATImage creates an empty DOS FAT file system image in file_name. The label must not exceed 11 chars. cluster_size is given in numbers of sectors per cluster (typically 1 for floppy disks).

    The file system created will have only 1 FAT (usually there are 2) and only a minimum number of root directory entries. This way we save 11 kbytes on a 1.44M floppy.

    Note: you can specify such things as a cluster_size of 5 sectors/cluster. Linux won't choke on it. But I don't know how Win/Dos will like it.

    Return Values

    MakeFATImage returns a list with 2 elements:

    (blocks, block_size)

    • blocks is the number of blocks that can be used on that file system

    • block_size is the size of a block in bytes

    So, blocks * block_size gives the usable file system size in bytes.

    On any failure, ( ) is returned.
