Boot Disk Creation

These are some perl scripts to assist in making a boot floppy/cd:

    mk_initrd
    mk_boot
    mk_root
    mk_yast2
    mk_yast2_cd
    mk_yast2_nfs

    mk_initrd_test
    initrd_test

The scripts need some perl functions provided in these libraries:

    AddFiles
    ReadConfig
    MakeFATImage
    MakeMinixImage
You do not have to run these scripts manually. There is actually a Makefile that does everything for you:

    make
      - make everything except the initrd test
    make initrd
      - make the initrd
    make boot
      - make a boot floppy image
    make root
      - make a root image suitable for YaST2
    make yast2
      - make an image with all the YaST2 stuff on it (including X11)
    make yast2_cd
      - make a bootable YaSt2 test cd
    make yast2_nfs
      - setup a NFS installable YaST2
    make initrd_test
      - make the initrd test environment

    make html

      - update the documentation
To set up a NFS installable YaST2 put your NFS directory you want to use in etc/config and then do a "make nfs" and copy images/boot to a floppy.
