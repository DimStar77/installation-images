MakeMinixImage

MakeMinixImage.pm is a perl module that can be used to create Minix file systems. It exports the following symbols:

  • MakeMinixImage(file_name, size_in_kbyte, inodes)

Usage

use MakeMinixImage;

Description

  • MakeMinixImage(file_name, size_in_kbyte, inodes)

    MakeMinixImage creates an empty Minix file system image in file_name. size_in_kbyte is the size of the image. inodes is the number of inodes the filesystem should have at least.

    Return Values

    MakeMinixImage returns a list with 3 elements:

    (blocks, block_size, inodes)

    • blocks is the number of blocks that can be used on that file system

    • block_size is the size of a block in bytes

    • inodes is the actual number of usable inodes

    So, blocks * block_size gives the usable file system size in bytes.

    On any failure, ( ) is returned.
