mk_yast2

mk_yast2 creates a yast2 image (not compressed) that should be mounted to /usr within the root image and contains everything necessary to run YaST2.

This image is intended to go onto a bootable CD.

The files to go into the YaST2 image are taken from data/yast2/yast2.file_list.

The final image is stored as images/yast2.
