AddFiles

AddFiles.pm is a perl module that can be used to extract files from rpms. It exports the following symbols:

  • AddFiles(dir, file_list, ext_dir)

Usage

use AddFiles;

Description

  • AddFiles(dir, file_list, ext_dir)

    AddFiles extracts the files in file_list and puts them into dir. Files that are not to be taken from rpms are copied from ext_dir.

    The syntax of the file list is rather simple; please have a look at those provided with this package to see how it works. A syntax description follows later...

    On any failure, exit( ) is called.
